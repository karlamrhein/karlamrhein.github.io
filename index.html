1. Bash Aliases for Productivity
Speed up command-line work by defining aliases in ~/.bashrc or ~/.zshrc.
Example: 
alias ll='ls -alF'
alias gs='git status'
alias ..='cd ..'
Reload your shell config: source ~/.bashrc
2. Persistent SSH Connections (ControlMaster)
Reuse SSH connections for faster access. Add to ~/.ssh/config: 
Host *
  ControlMaster auto
  ControlPath ~/.ssh/cm-%r@%h:%p
  ControlPersist 10m
This keeps connections alive for 10 minutes.
3. Efficient File Searching with fzf
Fzf provides fuzzy file search. Install it (sudo apt install fzf) and use: 
fzf
Or: 
find . -type f | fzf
Bind to Ctrl+T for quick search in your shell config.
4. Customizing .bashrc & .zshrc
Add environment variables, PATH updates, and functions: 
export PATH="$HOME/bin:$PATH"
export EDITOR=vim
myfunc() { echo "Hello $1"; }
Apply changes: source ~/.bashrc
5. Systemd Service Management
Manage services with systemctl: 
systemctl status nginx
systemctl restart apache2
systemctl enable myservice
List all services: systemctl list-units --type=service
6. Network Traffic Monitoring with iftop & nload
Install tools: 
sudo apt install iftop nload
Run: 
sudo iftop -i eth0
sudo nload
See live network usage.
7. Disk Usage Analysis with ncdu
Install ncdu: 
sudo apt install ncdu
Analyze disk usage interactively: 
ncdu /
8. rsync for Efficient Backups
Use rsync for fast incremental backups: 
rsync -avh /source/dir/ /backup/dir/
For remote backups: 
rsync -avz /source/dir/ user@host:/backup/dir/
9. SSH Key Management
Generate a key: 
ssh-keygen -t ed25519 -C "your_email@example.com"
Add to agent: 
ssh-add ~/.ssh/id_ed25519
List keys: 
ssh-add -l
10. Setting Up UFW Firewall
Enable and configure firewall: 
sudo ufw enable
sudo ufw allow ssh
sudo ufw allow 80/tcp
sudo ufw status
11. Cron Scheduling Best Practices
Edit your crontab: 
crontab -e
List jobs: 
crontab -l
Example entry (run backup daily at 2am): 
0 2 * * * /usr/local/bin/backup.sh
12. Log Rotation with logrotate
Custom config in /etc/logrotate.d/myapp: 
/var/log/myapp/*.log {
  daily
  rotate 7
  compress
  missingok
  notifempty
}
Test logrotate: 
logrotate -d /etc/logrotate.conf
13. Top & htop for Resource Monitoring
top
Install htop: 
sudo apt install htop
htop
htop provides color, filtering, and process tree.
14. tmux for Terminal Multiplexing
Start tmux: 
tmux
New session: tmux new -s mysession
Split panes: Ctrl+b % (vertical), Ctrl+b " (horizontal) Detach: Ctrl+b d, List sessions: tmux ls
15. Customizing Prompt with PS1
Example prompt (add to ~/.bashrc): 
export PS1="\u@\h \W \$ "
Add color: 
export PS1="\[\e[32m\]\u@\h:\[\e[34m\]\W\[\e[m\] \$ "
16. Using xargs for Powerful Command Pipelines
Delete all log files: 
find . -name "*.log" | xargs rm
Run commands in parallel: 
cat hosts.txt | xargs -n1 -P4 ping -c1
17. Monitoring Services with Monit
Install monit: 
sudo apt install monit
Configure /etc/monit/monitrc: 
check process sshd with pidfile /var/run/sshd.pid
  start program "/etc/init.d/ssh start"
  stop program "/etc/init.d/ssh stop"
  if failed port 22 protocol ssh then restart
  if 5 restarts within 5 cycles then timeout
Start monit: sudo monit
18. Automounting Drives with fstab
Example /etc/fstab entry: 
/dev/sdb1 /mnt/data ext4 defaults 0 2
Test mounts: 
sudo mount -a
19. Secure File Deletion with shred
Overwrite and delete: 
shred -u -v filename.txt
3 passes by default, use -n to set number.
20. System Resource Limits with ulimit
Check current limits: 
ulimit -a
Set open file limit: 
ulimit -n 4096
For permanent changes, edit /etc/security/limits.conf.
21. Tracing Processes with strace
Attach to running process: 
strace -p 1234
Trace command: 
strace ls -l
Filter syscalls: 
strace -e open,read,write ls
22. Using lsof for Open File Tracking
List all open files: 
lsof
Check files by process: 
lsof -p 1234
Network connections: 
lsof -i :80
23. Process Management with nice & renice
Start a process with lower priority: 
nice -n 10 mycommand
Change priority of running process: 
renice -n 15 -p 1234
24. Custom Kernel Parameters (sysctl)
Apply parameter temporarily: 
sudo sysctl -w net.ipv4.ip_forward=1
Make permanent by editing /etc/sysctl.conf: 
net.ipv4.ip_forward = 1
Reload settings: 
sudo sysctl -p
25. Efficient File Compression (tar, gzip, bzip2, xz)
Create tar.gz archive: 
tar czvf archive.tar.gz /folder
Extract: 
tar xzvf archive.tar.gz
Use bzip2 or xz for better compression: 
tar cjvf archive.tar.bz2 /folder
tar cJvf archive.tar.xz /folder
26. Dynamic DNS with ddclient
Install and configure ddclient for dynamic DNS updates (/etc/ddclient.conf): 
protocol=dyndns2
use=web, web=checkip.dyndns.com
server=members.dyndns.org
login=yourusername
password=yourpassword
yourhost.dyndns.org
Start ddclient: sudo systemctl start ddclient
27. Monitoring Disk SMART Status
Install smartmontools: 
sudo apt install smartmontools
Check health: 
sudo smartctl -a /dev/sda
Run test: 
sudo smartctl -t short /dev/sda
28. Using awk for Text Processing
Print first column: 
awk '{print $1}' file.txt
Sum values: 
awk '{s+=$2} END {print s}' data.txt
Filter by pattern: 
awk '/error/ {print $0}' logfile
29. File Permissions & ACLs
Change permissions: 
chmod 644 file.txt
Change owner: 
chown user:group file.txt
Add ACL for user: 
setfacl -m u:anotheruser:rwx file.txt
30. Automating Tasks with Ansible
Install ansible: 
sudo apt install ansible
Simple playbook (playbook.yml): 
- hosts: web
  tasks:
    - name: Install nginx
      apt: name=nginx state=present
Run playbook: 
ansible-playbook playbook.yml -i inventory
31. Setting up SSH Tunnels
Forward local port to remote: 
ssh -L 8080:localhost:80 user@server
Remote port forwarding: 
ssh -R 9090:localhost:22 user@server
32. Efficient Grep Usage
Search recursively: 
grep -r "pattern" .
Highlight matches: 
grep --color=auto "pattern" file.txt
Exclude files: 
grep --exclude=*.log "pattern" *
33. Custom DNS Resolver Configuration
Edit /etc/resolv.conf: 
nameserver 8.8.8.8
nameserver 1.1.1.1
For systemd-resolved: 
sudo systemctl status systemd-resolved
sudo resolvectl dns eth0 8.8.8.8 1.1.1.1
34. Viewing Hardware Info (lshw, lsblk, lspci)
List hardware: 
sudo lshw -short
lsblk
lspci
lsusb
35. Configuring syslog for Centralized Logging
Edit /etc/rsyslog.conf to send logs to server: 
*.* @logserver.example.com:514
Restart rsyslog: 
sudo systemctl restart rsyslog
36. Using sed for Stream Editing
Replace text in file: 
sed -i 's/foo/bar/g' file.txt
Delete lines containing pattern: 
sed -i '/pattern/d' file.txt
37. Checking Network Ports with netstat & ss
List open ports: 
netstat -tuln
ss -tuln
Show processes using ports: 
sudo lsof -i -P -n | grep LISTEN
38. Running Commands in Background (&, nohup)
Run in background: 
mycommand &
Keep running after logout: 
nohup mycommand &
39. Package Management Shortcuts (apt, yum, dnf)
Update and upgrade: 
sudo apt update && sudo apt upgrade
Install package: 
sudo yum install nginx
sudo dnf install nginx
Remove package: 
sudo apt remove nginx
40. Managing Swap Space
Create swap file: 
sudo fallocate -l 1G /swapfile
sudo chmod 600 /swapfile
sudo mkswap /swapfile
sudo swapon /swapfile
Make permanent by adding to /etc/fstab: 
/swapfile none swap sw 0 0
41. Setting up Fail2Ban for Intrusion Prevention
Install: 
sudo apt install fail2ban
Configure jail: 
[sshd]
enabled = true
port    = ssh
filter  = sshd
logpath = /var/log/auth.log
maxretry = 5
Restart fail2ban: 
sudo systemctl restart fail2ban
42. Encrypting Files with GnuPG
Encrypt file: 
gpg -c file.txt
Decrypt file: 
gpg -d file.txt.gpg > file.txt
43. Configuring RAID with mdadm
Create RAID 1: 
sudo mdadm --create /dev/md0 --level=1 --raid-devices=2 /dev/sda1 /dev/sdb1
View status: 
cat /proc/mdstat
Save config: 
sudo mdadm --detail --scan | sudo tee -a /etc/mdadm/mdadm.conf
44. Monitoring Resources with glances
Install glances: 
sudo apt install glances
Run: 
glances
Web mode: 
glances -w
Access via browser: http://localhost:61208
45. Setting up SSH Agent Forwarding
Enable agent forwarding in ~/.ssh/config: 
Host server
  ForwardAgent yes
Connect: 
ssh server
Now, keys from your local agent are available remotely.
46. Customizing System Boot with GRUB
Edit /etc/default/grub: 
GRUB_TIMEOUT=5
GRUB_CMDLINE_LINUX_DEFAULT="quiet splash"
Apply changes: 
sudo update-grub
47. Using screen for Persistent Sessions
Start screen: 
screen
Detach: Ctrl+a d
List sessions: 
screen -ls
Reattach: 
screen -r
48. Monitoring System Logs with journalctl
View logs: 
journalctl -xe
Logs for a service: 
journalctl -u nginx
Follow logs: 
journalctl -f
49. Limiting Resource Usage with cgroups
Create cgroup: 
sudo cgcreate -g memory,cpu:mygroup
Limit memory: 
echo 256M | sudo tee /sys/fs/cgroup/memory/mygroup/memory.limit_in_bytes
Run process in cgroup: 
sudo cgexec -g memory,cpu:mygroup mycommand
50. Setting Up Automated Alerts (mail, notify-send)
Send email alert (install mailutils): 
echo "Disk full!" | mail -s "Alert" user@example.com
Desktop notification: 
notify-send "Backup completed!"
Use in scripts to automate system alerts.
